# Nebyss Innovation ‚Äî Nuxt Landing Page

## Project Overview
**Nebyss Innovation** (`nebyss.eu`) ‚Äî a European AI company building domain-specific, human-supervised intelligence. This is the Nuxt 3 replacement for the original WordPress + Divi site, built for full design control, performance, and animation quality.

## Tech Stack
- **Framework:** Nuxt 3 (Vue 3, Nitro server engine)
- **Rendering:** SSG via `nuxt generate` (static site generation)
- **Styling:** Plain CSS with custom properties (design tokens). No Tailwind, no CSS framework.
- **Fonts:** Inter (Google Fonts), loaded via `<link>` in `nuxt.config.ts`
- **Animations:** CSS transitions + Intersection Observer (composable). No animation library.
- **Deployment target:** Static hosting (Cloudflare Pages, Vercel, Netlify, etc.)

## Design Language
The design follows **Anthropic's design philosophy** + **Lazy Dynamics-inspired** visual elements:
- **Warm dark palette** ‚Äî `#0c0c0c` base (not pure black), warm off-white text `#e8e4de`
- **Restrained typography** ‚Äî Inter at weight 400-600, never 800. Feature settings `cv02, cv03, cv04, cv11`
- **Generous whitespace** ‚Äî 100px section padding, 680px max text width
- **Subtle interactions** ‚Äî 0.7s reveals with `cubic-bezier(0.16, 1, 0.3, 1)`, 3.5% opacity cursor glow
- **Editorial feel** ‚Äî eyebrow labels, gradient text, accent pills, card top-edge highlights
- **Accent color:** Terra cotta `#c9856b` (warm, approachable, not aggressive)
- **Layered logo box** ‚Äî hero has a nested depth box with animated gradient border + corner brackets (LD-inspired)
- **Feature pillars** ‚Äî 4-column strip with vertical accent bars below hero (LD-inspired)
- **Animated gradient borders** ‚Äî `.nb-card--gradient` variant with shifting gradient outline
- **Light CTA break** ‚Äî `#f5f2ed` warm-light section before footer for contrast rhythm
- **Rounded CTA button** ‚Äî dark pill-shaped button on light section (LD-inspired)

## CSS Class System
- `.nb-container` ‚Äî centered max-width container with padding
- `.nb-section` ‚Äî section with 100px padding + bottom border
- `.nb-section--alt` ‚Äî alternate background (`--nb-bg-2`)
- `.nb-grid`, `.nb-grid--2`, `.nb-grid--3`, `.nb-grid--60-40` ‚Äî CSS grid layouts
- `.nb-card`, `.nb-card--sm` ‚Äî elevated card with top-edge highlight + hover lift
- `.nb-btn`, `.nb-btn--primary` ‚Äî buttons with shimmer hover effect
- `.nb-eyebrow` ‚Äî uppercase small label
- `.nb-pill` ‚Äî accent-colored pill badge
- `.nb-reveal` ‚Äî scroll reveal target (add `data-delay="1-4"` for stagger)

## Animation System
### Scroll Reveal (`useScrollReveal`)
- Targets all `.nb-reveal` elements via Intersection Observer
- Threshold: 0.12, rootMargin: -60px
- Adds `.is-visible` class which triggers CSS transition (0.7s spring ease)
- Supports `data-delay` attributes for stagger (0.06s, 0.14s, 0.22s, 0.32s)
- Direction variants: `.nb-reveal--left`, `.nb-reveal--right`, `.nb-reveal--scale`
- Respects `prefers-reduced-motion`

### Cursor Glow (`useCursorGlow`)
- 400px radial gradient, 3.5% opacity terra cotta
- Lerp smoothing at 0.06 (very soft, organic follow)
- Only activates on `pointer: fine` devices (no touch)
- Appends `#nb-cursor-glow` to body, cleans up on unmount

### Progress Bar
- Built into `NbHeader.vue`, 1px height
- Gradient from 50% to 25% accent opacity
- Updates on scroll via passive listener

## Key Conventions
1. **Component naming:** `Nb` prefix (NbHeader, NbHero, etc.) to avoid conflicts
2. **Scoped styles:** Every component uses `<style scoped>` for isolation
3. **No `!important`:** CSS specificity handled via proper scoping ‚Äî no Divi overrides needed
4. **Composable pattern:** Auto-imported from `composables/` directory
5. **Static generation:** All pages pre-rendered via `routeRules` in `nuxt.config.ts`
6. **No external JS libs:** All animations pure CSS + vanilla JS composables
7. **Semantic HTML:** `<section>`, `<article>`, `<nav>`, `<address>`, `<main>`, `<footer>`

## Adding New Sections
1. Create `components/NbSectionName.vue` with `<template>`, `<script setup>`, `<style scoped>`
2. Use `.nb-section` wrapper, `.nb-container` for content, `.nb-reveal` for animations
3. Import in `pages/index.vue` (auto-imported, just add `<NbSectionName />`)

## Adding New Pages
1. Create `pages/page-name.vue`
2. Nuxt auto-generates the route at `/page-name`
3. Add `useScrollReveal()` in `<script setup>` for scroll animations
4. Link from header/footer navigation

## Deployment
```bash
npm run generate                      # build static
npx wrangler pages deploy .output/public  # Cloudflare Pages
# or drag .output/public/ to Vercel/Netlify
```

## Performance Notes
- Zero JS animation libraries ‚Äî all CSS transitions
- Intersection Observer disconnects after element revealed (one-shot)
- Cursor glow only on desktop (pointer:fine check)
- Font loaded via preconnect + display=swap
- All images should use `<NuxtImg>` or native lazy loading when added
- Target: 95+ Lighthouse score on all metrics
